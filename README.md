Task Manager Fullstack

A fullstack Task Manager application built with Node.js and Express on the backend and a simple frontend to manage tasks.

This project demonstrates a complete CRUD API and basic fullstack structure, focused on clean code, organization, and real-world backend practices.

Features

- Create tasks
- List all tasks
- Update tasks
- Delete tasks
- RESTful API with Express
- Data persistence using JSON file
- Organized routes structure

Tech Stack

Backend
- Node.js
- Express.js
- File System (fs)
- REST API

Frontend
- HTML
- CSS
- JavaScript
